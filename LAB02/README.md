Part 1 : Dataset Exploration (LAB2-1.py)

เริ่มต้นด้วยการนำเข้าไลบรารี Pandas เพื่อใช้จัดการข้อมูล จากนั้นโหลดชุดข้อมูล fifa.csv เข้ามาเก็บไว้ในตัวแปร df โดยใช้คำสั่ง pd.read_csv()

หลังจากโหลดข้อมูลแล้ว จะทำการสำรวจข้อมูลเบื้องต้น ดังนี้

ตรวจสอบขนาดของข้อมูลด้วย df.shape เพื่อดูจำนวนแถว (Rows) และจำนวนคอลัมน์ (Columns)
ตรวจสอบชนิดข้อมูลของแต่ละคอลัมน์ด้วย df.dtypes เพื่อดูว่าข้อมูลเป็นตัวเลข ข้อความ หรือประเภทอื่น
แสดงค่าสถิติพื้นฐานของข้อมูลด้วย df.describe(include="all") เช่น ค่าเฉลี่ย ค่ามากที่สุด ค่าน้อยที่สุด และจำนวนข้อมูล
ตรวจสอบค่าที่หายไป (Missing Values) ด้วย df.isnull().sum() เพื่อดูว่าคอลัมน์ใดมีข้อมูลไม่ครบ
ตรวจสอบข้อมูลที่ซ้ำกัน (Duplicate Records) ด้วย df.duplicated().sum()
ตรวจสอบการกระจายของตำแหน่งนักฟุตบอล (Position) ด้วย df["position"].value_counts() เพื่อดูว่ามีผู้เล่นในแต่ละตำแหน่งจำนวนเท่าใด

ขั้นตอนนี้มีจุดประสงค์เพื่อทำความเข้าใจข้อมูลก่อนนำไปวิเคราะห์และประมวลผลต่อไป

Part 2 : Data Visualization (LAB2-2.py)

หลังจากสำรวจข้อมูลแล้ว จะทำการแสดงผลข้อมูลในรูปแบบกราฟ เพื่อให้เห็นลักษณะของข้อมูลได้ชัดเจนยิ่งขึ้น

เริ่มจากการนำเข้าไลบรารี Matplotlib และ Seaborn

จากนั้นสร้าง Histogram ของคอลัมน์ age เพื่อดูการกระจายตัวของอายุผู้เล่น ว่าผู้เล่นส่วนใหญ่อยู่ในช่วงอายุใด

ต่อมาจะสร้าง Correlation Heatmap โดยเลือกเฉพาะข้อมูลประเภทตัวเลข แล้วคำนวณค่าความสัมพันธ์ (Correlation) ระหว่างตัวแปร จากนั้นใช้ sns.heatmap() แสดงผลเป็นแผนภาพสี ซึ่งช่วยให้เห็นว่าตัวแปรใดมีความสัมพันธ์กันมากหรือน้อย

ขั้นตอนนี้ช่วยให้สามารถมองเห็นรูปแบบของข้อมูลและความสัมพันธ์ระหว่างตัวแปรต่าง ๆ ได้ง่ายขึ้น

Part 3 : Data Cleaning (LAB2-3.py)

ในขั้นตอนนี้เป็นการทำความสะอาดข้อมูล (Data Cleaning) เพื่อให้ข้อมูลมีคุณภาพและพร้อมสำหรับการนำไปสร้างโมเดล Machine Learning

เริ่มจากการสร้างสำเนาของข้อมูล เพื่อเปรียบเทียบผลของการเติมค่าที่หายไปด้วย Mean และ Median

วิธีแรกคือการแทนค่าที่หายไปของข้อมูลตัวเลขด้วยค่าเฉลี่ย (Mean) โดยเลือกเฉพาะคอลัมน์ที่เป็นตัวเลข แล้วใช้ fillna() เติมค่าที่หายไป

วิธีที่สองคือการแทนค่าที่หายไปด้วยค่ามัธยฐาน (Median) ซึ่งเหมาะกับข้อมูลที่มีค่าผิดปกติ (Outliers)

จากนั้นตรวจสอบข้อมูลซ้ำ โดยใช้ duplicated().sum() แล้วลบข้อมูลที่ซ้ำด้วย drop_duplicates() พร้อมตรวจสอบจำนวนข้อมูลหลังลบอีกครั้ง

ต่อมาจะตรวจสอบข้อมูลที่ผิดปกติ เช่น อายุที่มีค่าติดลบ หากพบจะเปลี่ยนค่าเป็นค่ามัธยฐานของอายุ เพื่อให้ข้อมูลมีความสมเหตุสมผล

สุดท้ายตรวจสอบชนิดข้อมูลอีกครั้ง และเปรียบเทียบผลของการเติม Missing Values ด้วย Mean และ Median เพื่อดูความแตกต่างของทั้งสองวิธี

Part 4 : Feature Engineering (LAB2-4.py)

ขั้นตอนสุดท้ายคือการเตรียมข้อมูลให้อยู่ในรูปแบบที่โมเดล Machine Learning สามารถนำไปใช้งานได้

เริ่มจากการใช้ Label Encoding เพื่อแปลงข้อมูลประเภทข้อความในคอลัมน์ preferred_foot เช่น Left และ Right ให้กลายเป็นค่าตัวเลข เช่น 0 และ 1

หลังจากนั้นใช้ One-Hot Encoding กับคอลัมน์ position โดยใช้คำสั่ง pd.get_dummies() เพื่อแยกแต่ละตำแหน่งของนักฟุตบอลออกเป็นคอลัมน์ใหม่ เช่น position_FW, position_MF, position_DF และ position_GK ซึ่งแต่ละคอลัมน์จะเก็บค่า 0 หรือ 1 แทนการมีหรือไม่มีตำแหน่งนั้น

เมื่อทำการเข้ารหัสข้อมูลเรียบร้อยแล้ว จะแสดงข้อมูลตัวอย่างและตรวจสอบขนาดของชุดข้อมูลใหม่ เพื่อยืนยันว่าการแปลงข้อมูลเสร็จสมบูรณ์
